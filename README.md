# 🎉 EventFlow - Event Management System

A Full Stack Event Management System built with React, Node.js, Express, MongoDB and Tailwind CSS.

---

## 📖 Overview

EventFlow is a web application that helps event companies manage the entire event organization process from receiving customer requests to successfully completing events.

The system digitizes the traditional event planning workflow, helping companies reduce paperwork, improve collaboration, and provide better customer experiences.

---

## ✨ Main Features

- User Authentication & Authorization
- Event Request Management
- Event Planning
- Task Management
- Design Review & Approval
- Quotation Management
- Payment Tracking
- Notification System
- Customer Reviews
- Dashboard & Analytics
